# SQS 参数设置指南

## 概述

Special Quasirandom Structure (SQS) 是一种用于模拟随机合金或掺杂系统的周期性结构。
本指南帮助你理解每个参数的含义和推荐设置。

---

## 一、 ClusterSpace 参数

### 1.1 cutoffs (团簇截断半径)

**含义**: 定义团簇展开的截断距离，单位 Å

**默认值**: `[5.0]`

**推荐范围**:
- 小系统 (≤50原子): 4.0 - 5.0 Å
- 中等系统 (50-200原子): 5.0 - 7.0 Å
- 大系统 (>200原子): 7.0 - 10.0 Å

**影响**:
- **太小** (<4.0Å): 
  - 只考虑最近邻团簇
  - 可能丢失重要的远程相互作用
  - SQS 质量可能不佳
  
- **适中** (5.0-7.0Å): ✓ 推荐
  - 平衡计算成本和精度
  - 包含足够的团簇信息
  - 适合大多数系统
  
- **太大** (>10.0Å):
  - 计算时间大幅增加 (指数级增长)
  - 可能过拟合
  - 内存消耗增加

**设置建议**:
```python
# 簡单氧化物系统
"cutoffs": [5.0]

# 复杂金属合金
"cutoffs": [7.0]

# 非常精确的计算
"cutoffs": [8.0, 5.0]  # 多尺度截断
```

### 1.2 symprec (对称性容差)

**含义**: 用于识别晶体对称性的容差

**默认值**: `1e-3`

**推荐范围**: `1e-5` 到 `1e-2`

**影响**:
- **太小** (1e-6): 对原子位置要求过于严格，可能丢失对称性
- **适中** (1e-3 到 1e-4): ✓ 推荐，适合大多数情况
- **太大** (>1e-2): 可能错误地识别对称性，导致结构约束不正确

**设置建议**:
- 高对称性结构 (立方、四方): 使用 1e-3
- 低对称性结构: 使用 1e-4
- 不确定时: 先用 1e-3，如失败则放宽

---

### 1.3 position_tolerance (位置容差)

**含义**: 判断原子位置等价性的容差

**默认值**: `0.1`

**推荐范围**: `0.05` 到 `0.2`

**影响**:
- **太小** (<0.05): 对原子位置要求过严，可能导致等价位点被误判为不同位点
- **适中** (0.1): ✓ 推荐
- **太大** (>0.2): 可能将不同位点错误地合并

## 二、 SQS 生成参数 - 通用

### 2.1 tolerance (目标偏差)

**含义**: SQS 团簇向量与理想随机合金的允许偏差

**默认值**: `0.001`

**推荐范围**:
- 高质量 SQS: `0.001` (优秀标准)
- 可接受质量: `0.01` (良好标准)
- 快速测试: `0.1` (可用标准)

**影响**:
- **太小** (0.0001): 可能无法达到，导致无限循环
- **适中** (0.001-0.01): ✓ 推荐，平衡质量和计算时间
- **太大** (>0.1): SQS 质量较差，团簇函数偏差大

**与原子数的关系**:
| 无序原子数 | 推荐最小偏差 | 说明 |
|----------|------------|------|
| 10-20 | 0.01-0.1 | 小系统物理限制 |
| 20-50 | 0.005-0.01 | 中等系统 |
| 50-200 | 0.001-0.005 | 理想范围 |
| >200 | 0.001 | 可达到理想SQS |

**van de Walle 2013 质量标准**:
- **优秀** ✅: deviation < 0.001
- **良好** ✓: deviation < 0.01
- **可用**: deviation < 0.10
- **可接受** (有限系统): deviation < 0.30
- **失败** ❌: deviation ≥ 0.30

---

### 2.2 random_seed (随机种子)

**含义**: 控制随机数生成器的种子，用于结果可重复

**默认值**: `None` (随机)

**推荐设置**:
- **调试/测试**: 设置固定值 (如 `42`)
- **生产环境**: 使用 `None` (随机)
- **需要重现结果**: 记录并复用相同的种子

**影响**:
- **None**: 每次运行结果可能不同
- **固定值**: 每次运行结果完全相同 (便于调试和比较)

**使用场景**:
```python
# 场景1: 调试时固定结果
"random_seed": 42

# 场景2: 生产环境随机
"random_seed": null

# 场景3: 比较不同方法
# 方法A 使用 seed=123
# 方法B 使用 seed=123 (公平比较)
```

---

### 2.3 pbc (周期边界条件)

**含义**: 三个方向的周期性边界条件

**默认值**: `[True, True, True]`

**推荐设置**:
- **三维块体材料**: `[True, True, True]` ✓
- **二维材料 (薄膜)**: `[True, True, False]`
- **表面/界面**: `[True, False, False]`
- **纳米线**: `[True, True, False]`

**影响**:
- **全部 True**: 完整的周期性结构
- **部分 False**: 对应方向为非周期性 (表面、薄膜等)

**注意事项**:
- 确保与你的物理系统匹配
- 错误的 pbc 会导致错误的 SQS 结构
